This is a change made in the local dev environment that will now move to test

Just like pipeline stages, environment setups vary. The following setup is common:

  • dev: a development-only place for adding new code, kept locally on a developer's computer
  • test: a testing-only place for verifying that code is good enough to publish, hosted on a private server
  • prod: the live "production" version of a product, hosted online for users to see
